# 🔧 Developer Toolbox — VS Code Extension

> **All-in-one developer utilities — 18+ tools running directly inside VS Code!**

No browser needed. No sign-ups. No data collection. Everything runs client-side inside a VS Code webview panel.

## ✨ Features

### 🛠️ Developer Tools (7)
- **README Generator** — Generate professional README.md with templates & scoring
- **API Tester** — Test REST & WebSocket APIs with environments & assertions
- **Mock API Generator** — Generate fake REST APIs with realistic data
- **JWT Toolkit** — Decode, verify, build & audit JSON Web Tokens
- **JSON Formatter** — Format, validate, minify & transform JSON data
- **Regex Generator** — Build & test regex patterns with live preview
- **Password Generator** — Generate secure passwords, passphrases & PINs

### 🎨 Frontend Tools (6)
- **Color Palette** — Generate & export beautiful color palettes
- **CSS Gradient** — Create CSS gradients with live preview
- **Box Shadow** — Create layered box shadows visually
- **Glassmorphism** — Create frosted glass UI effects
- **Grid Generator** — Click & drag CSS Grid layouts
- **Frontend Playground** — Code HTML, CSS & JS with live preview

### 📚 Learning Tools (5)
- **Sorting Visualizer** — Learn sorting algorithms visually
- **Recursion Visualizer** — Visualize recursive calls
- **Event Loop Visualizer** — See JS event loop in action
- **Flexbox Playground** — Learn CSS Flexbox visually
- **SQL Playground** — Practice SQL queries in-browser

## 🚀 Usage

- **Command Palette**: `Ctrl+Shift+P` → `Developer Toolbox: Open`
- **Keyboard Shortcut**: `Ctrl+Shift+T` (`Cmd+Shift+T` on Mac)
- **Activity Bar**: Click the 🔧 icon in the sidebar
